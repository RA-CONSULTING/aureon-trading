## Project Title

Aureon Rapid Evidence Evaluation Sprint

## Short Summary

R&A Consulting and Brokerage Services Ltd requests 20,000 AIRR GPU hours for a three-month rapid evaluation sprint to benchmark Aureon Evidence OS: a local-first, auditable AI workflow layer that converts research material into validated, grant-ready evidence packs. The project will run reproducible experiments on public, synthetic, and applicant-owned non-sensitive data to measure source classification, evidence routing, extraction quality, hallucination controls, and report-generation reliability. It will not process private Gmail or Drive content, live trading data, child-level SEN data, medical records, or other sensitive personal data.

## Objectives

1. Build a compact public benchmark suite for evidence extraction and grant-readiness classification.
2. Compare baseline retrieval and summarisation workflows against Aureon evidence-routing controls.
3. Run ablation studies across prompts, models, seed settings, and validation gates.
4. Produce an open technical report, reproducible scripts, and a demonstration package.

## Why AIRR Is Needed

Local hardware can run the operator repo and small prototypes, but it cannot run repeated benchmark sweeps across enough model, seed, and dataset combinations to produce credible evidence. The rapid route is proportionate because the sprint is focused on a narrow, auditable benchmark rather than the wider 12-month Open Access programme already submitted.
