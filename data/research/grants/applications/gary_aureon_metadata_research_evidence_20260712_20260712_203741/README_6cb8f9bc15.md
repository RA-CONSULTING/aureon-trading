# Funder-Ready Supporting Evidence

This folder contains polished documents intended for grant portals, funders, advisers, and partners.

Use these files when an application asks for supporting evidence, a project summary, a validation plan, or an applicant capability note.

Files outside this folder may include operational receipts, portal logs, automation status, risk controls, and internal working notes. Those records are useful for Aureon operations but should not be attached to funder-facing applications unless converted into a formal document first.
