## Public Description

R&A Consulting and Brokerage Services Ltd will contribute Aureon Evidence OS as an evidence automation and assurance work package for a clean maritime innovation project. The work package will help the project team organise source materials, maintain reproducible workflow records, classify project claims, prepare trial or deployment evidence, and produce funder-ready audit trails.

## Project Summary

The project will use Aureon Evidence OS to organise evidence for 100% battery electric vessels, charging infrastructure, shore power systems, or vessel upgrades. Aureon will provide an assurance work package within the consortium, using partner-supplied vessel, site, infrastructure, budget, permit, safety, and deployment evidence where those elements are in scope.

## Innovation

Aureon Evidence OS is a local-first approach to evidence management for innovation projects. It is designed to improve how project teams collect, organise, verify, and report technical evidence during research, trial, and deployment activity. The proposed work package focuses on source inventories, traceable claims, reproducible records, risk tagging, and structured reporting outputs that can support partner review and funder assessment.

## Project Need

Clean maritime innovation projects generate evidence across documents, test records, safety notes, partner reviews, compliance material, capital plans, deployment logs, and operational systems. Without a controlled evidence process, project teams can lose traceability between technical claims, delivery activity, and the material needed for review. Aureon addresses this need by keeping project claims connected to source records and preparing clear reporting outputs for partners and funders.

## Aureon Role

Aureon provides evidence automation and assurance for clean maritime trial, deployment, safety, compliance, or operations reporting. The work package covers evidence source inventory, claim classification, source mapping, reproducible workflow records, technical evidence pack preparation, risk and assumption tagging, partner review materials, and funder-ready audit trails.
