Phone Setup
  1. Connect the phone to the same WiFi this computer is on.
  2. Open the URL above in the phone's browser (Safari / Chrome / Edge).
  3. Tap Share → Add to Home Screen. The Aureon tile becomes a real launcher on the phone.
  4. Open the tile. The status strip turns green when the phi-bridge is coupled.
  5. Talk by holding the orange mic, or type to send.

Firewall
If the phone can see your WiFi but times out opening the URL, Windows Firewall is probably blocking the port. Open an elevated PowerShell and run: 
New-NetFirewallRule -DisplayName 'Aureon Vault' `
  -Direction Inbound -Action Allow `
  -Protocol TCP -LocalPort 5566 -Profile Private
